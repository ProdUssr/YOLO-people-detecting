# Парсер фотографий с детекцией людей

## Описание
Проект для парсинга фотографий с сайта и детекции людей с помощью YOLO.

## Функциональность
- Парсинг фотографий с сайта
- Детекция людей с помощью YOLO
- Графический интерфейс для просмотра

## Инструкция по использиванию
Скачайте или клонируйте репозиторий к себе на ПК, приведу пример дальнейших действий для 
Windows: скачали проект например в папку C:\Projects, 
далее откройте редактор кода ( VScode, PyCharm и др. ) или обычный терминал. 
В терминале редактора кода или любого другого перейдите в папку проекта командой 
`cd C:\Projects\YOLO-people-detecting`,  после этого создайте 
виртуальное окружение для этого проекта, чтобы библиотеки не конфликтовали с другими 
библиотеками на вашем ПК, это делается командой `python -m venv YOLO_env`  
вместо YOLO_env может быть любое название
Далее активируйте виртуальное окружение командой `YOLO_env\Scripts\activate.bat` 
( если вы используете PowerShell, то в начале команды будет 
точка `.YOLO_env\Scripts\activate.bat` ), в терминале слева от пути 
должна появится приписка (YOLO_env)
Если всё получилось, то установите необходимые библиотеки командой 
`pip install -r requirements.txt`
После этого можно запускать файл с детекцией, для этого нажмите "Run" в редакторе кода или введите 
`python people_detecting.py`
Фотографии уже загружены в папку RSK, поэтому можно приступать к детекции, 
если хотите загрузить больше или меньше фотографий через парсер, то очистите 
папку RSK от фотографий и запустите парсер командой 
`python image_parser.py` или графическим интерфейсом в редакторе кода
